# Stripe Payment Gateway

## Overview
Stripe is one of the most popular payment processors worldwide, offering comprehensive payment solutions for online businesses.

## Features
- **Credit & Debit Cards**: Accept all major card types
- **Digital Wallets**: Apple Pay, Google Pay, etc.
- **Recurring Payments**: Subscriptions and scheduled payments
- **Refunds**: Full and partial refund support
- **Tokenization**: Save payment methods securely
- **Strong Customer Authentication (SCA)**: EU compliance
- **Multi-currency**: Support for 135+ currencies

## Configuration
1. Create a Stripe account at https://stripe.com
2. Get your API keys from the Stripe Dashboard
3. Configure webhook endpoints for payment notifications
4. Set up your business information for compliance

## Supported Countries
Available in 40+ countries including US, Canada, UK, EU, Australia, and more.
