# SEO Settings Documentation

**Location:** WordPress Admin → Yatra → Settings → SEO

---

## 🚀 SEO Overview

SEO (Search Engine Optimization) settings help your travel website rank better in search engines like Google, making it easier for potential customers to find your trips. Think of this as optimizing your online presence to attract more organic traffic and bookings.

---

## 📝 SEO Content Configuration

### SEO Trip Meta Title
- **Field Type:** Text Input
- **Default:** Empty
- **Description:** Template for trip page meta titles
- **What it does:** Controls the title that appears in search engine results
- **Where it appears:** Google search results, browser tabs, social media shares
- **Default behavior:** Uses trip title if not set
- **Template variables:** Use placeholders like `{trip_title}`, `{location}`, `{price}`
- **Best practices:**
  - **Length:** Keep under 60 characters for optimal display
  - **Keywords:** Include relevant travel keywords
  - **Branding:** Include your business name
  - **Compelling:** Make it clickable and attractive
- **Good examples:**
  - `{trip_title} | {location} Adventure Tours | Your Company`
  - `Book {trip_title} - {price} | {location} Travel Experiences`
  - `{trip_title} Trip | Expert Guides | Your Travel Company`
- **Why it matters:**
  - **Search ranking:** Meta titles are important ranking factors
  - **Click-through rate:** Good titles get more clicks
  - **Brand recognition:** Consistent title structure builds brand
- **⭐ Pro Tip:** Include location and price information when relevant

### SEO Trip Meta Description
- **Field Type:** Textarea
- **Default:** Empty
- **Description:** Template for trip page meta descriptions
- **What it does:** Sets the description that appears in search results
- **Where it appears:** Google search results, social media previews
- **Default behavior:** Uses trip excerpt if not set
- **Template variables:** Use placeholders like `{trip_title}`, `{duration}`, `{inclusions}`
- **Best practices:**
  - **Length:** Keep under 160 characters for optimal display
  - **Compelling:** Make users want to click and learn more
  - **Informative:** Include key trip details and benefits
  - **Call-to-action:** Encourage users to click or book
- **Good examples:**
  - `Experience {trip_title} in {location}. {duration} adventure including {inclusions}. Book now for early bird discounts!`
  - `Join our {trip_title} expedition. Expert guides, small groups, {duration} journey. Perfect for {difficulty_level} adventurers.`
  - `Discover {trip_title} - {location}'s most amazing experience. {duration} days of adventure, all equipment included.`
- **Why it matters:**
  - **Click-through rate:** Good descriptions get more clicks
  - **User expectations:** Sets clear expectations for trip content
  - **Search ranking:** Well-written descriptions can improve rankings
- **⭐ Pro Tip:** Include unique selling points and key differentiators

## 🎯 SEO Best Practices for Travel Websites

### 1. Keyword Strategy
- **Research keywords:** Understand what potential customers search for
- **Location-based terms:** Include city, region, country names
- **Activity keywords:** Use terms like "hiking," "diving," "safari"
- **Long-tail keywords:** Use specific phrases like "guided mountain trekking Nepal"

### 2. Content Optimization
- **Unique descriptions:** Each trip should have unique, compelling descriptions
- **Rich details:** Include practical information customers need
- **Visual content:** Use high-quality images and videos
- **User reviews:** Include customer testimonials and reviews

### 3. Technical SEO
- **URL structure:** Clean, descriptive URLs (covered in Permalink settings)
- **Mobile optimization:** Ensure site works perfectly on phones
- **Page speed:** Fast-loading pages rank better
- **Schema markup:** Use structured data for travel content

### 4. Local SEO
- **Google Business Profile:** Essential for local travel businesses
- **Local citations:** Ensure consistent business information online
- **Location pages:** Create pages for each location you serve
- **Customer reviews:** Encourage and respond to customer reviews

## 🔍 Advanced SEO Features

### Schema Markup (Structured Data)
- **What it is:** Special code that helps search engines understand your content
- **Travel schema:** Specific markup for travel-related content
- **Benefits:** Rich snippets in search results, better understanding
- **Implementation:** Often automatic with good SEO practices

### Open Graph Tags
- **What they are:** Meta tags for social media sharing
- **Benefits:** Better-looking social media previews
- **Implementation:** Usually automatic with good SEO setup
- **Importance:** Social sharing drives traffic and bookings

### XML Sitemaps
- **What they are:** Maps of your website structure for search engines
- **Benefits:** Better indexing, faster discovery of new content
- **Implementation:** Usually automatic with WordPress
- **Importance:** Essential for large travel websites

## ⚠️ Common SEO Issues and Solutions

### 1. Low Search Rankings
- **Problem:** Trips not appearing in search results
- **Causes:** Poor optimization, competition, technical issues
- **Solutions:** Improve content, build quality backlinks, fix technical issues
- **Prevention:** Regular SEO audits and monitoring

### 2. Poor Click-Through Rates
- **Problem:** Appearing in results but not getting clicks
- **Causes:** Poor titles/descriptions, low brand recognition
- **Solutions:** Improve meta titles and descriptions, build brand
- **Prevention:** A/B test different title/description approaches

### 3. Technical SEO Issues
- **Problem:** Site not being indexed properly
- **Causes:** Crawl errors, duplicate content, slow loading
- **Solutions:** Fix technical issues, improve site performance
- **Prevention:** Regular technical SEO audits

### 4. Local SEO Problems
- **Problem:** Not appearing in local search results
- **Causes:** Poor local optimization, inconsistent business information
- **Solutions:** Optimize Google Business Profile, build local citations
- **Prevention:** Maintain consistent local business information
